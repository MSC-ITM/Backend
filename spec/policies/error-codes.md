
# Error Codes

| code              | http | description                  |
|-------------------|------|------------------------------|
| VALIDATION_ERROR  | 400  | Datos inválidos              |
| NOT_FOUND         | 404  | Recurso no encontrado        |
| CONFLICT          | 409  | Conflicto de estado/versión  |
| RATE_LIMIT        | 429  | Límite de solicitudes        |
| INTERNAL          | 500  | Error inesperado             |
